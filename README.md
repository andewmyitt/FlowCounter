# FlowCounter
Код для микроконтроллера платы проточного счетчика и код для ПК
Код для платы лежит в архиве FlowCounter_v1.3, для редактирования нужно открыть файл FlowCounter_v1.3.ino. Файл RI_logo_image_120x120.h используется только для отображения логотипа института. Для редактирования кода удобнее всего пользоваться Arduino IDE 2.3.6. Чтобы загрузить для микроконтроллера нужно открутить нижние винты (8шт) и приподнять верхнуую крышку. Затем пальцем зажать и удерживать кнопку BOOT, подать питание на плату. Если все сделано верно экран будет гореть черным цветом. Необходимо подключить type-c, выбрать COM порт с нашим устройством в Arduino IDE, выбрать плату ESP32 dev module (необходима бибилотека плат esp32 от Espressif Systems) и нажать на загрузку кода на плату. Через пару минут появится надпись Hard Reset via Reset Pin это означает что код загружен и нужно нажать на кнопку RST (рядом с кнопкой BOOT). После этого микроконтроллер будет выполнять новый код. Если работа кода устраивает можно вытащить type-c, выключить питаниие и закрепить нижную часть с верхней на винты.

Для изменения кода утилиты для ПК потребуется Visual Studio Community (VSC) 2026. Код версии 1.3 лежит в zip архиве под названием Flow_Counter_Utility_v1.3. Нужно скачать архив, распаковать и открыть файл Flow_Counter_Utility.slnx с помощью VSC. Откроются 3 вкладки Form1.cs, Form1.Designer.cs и Form1.cs[Конструктор]. Визуал настраивается через  Form1.Designer.cs или Form1.cs[Конструктор], функционал в Form1.cs.
